# Exploitation

Press `Ctrl + U` on your browser to view the source code of the `HTML` file. Search for `flag{...}`. 

```html
<meta name="redpwnctf2020" content="flag{1nspector_g3n3ral_at_w0rk}">
```

The flag is:

```
flag{1nspector_g3n3ral_at_w0rk}
```
